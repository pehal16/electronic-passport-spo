# Экранная карта MVP

Проект: `Электронный паспорт группы СПО`

## 1. Общая структура навигации

### Публичные экраны

1. `Вход в систему`
2. `Восстановление доступа` (по email/телефону, опционально в MVP-lite)

### Приватные экраны (после авторизации)

1. `Главная панель` (роль-зависимая)
2. `Паспорт группы`
3. `Список студентов`
4. `Карточка студента`
5. `Журнал оценок`
6. `Посещаемость`
7. `Уведомления`
8. `Документы`
9. `Аналитика`
10. `Настройки и справочники` (admin)

## 2. Экраны по ролям

## 2.1 Администратор

1. Главная панель администратора
2. Пользователи и роли
3. Группы и специальности
4. Дисциплины и нагрузка преподавателей
5. Привязка родителя к студенту
6. Справочники
7. Отчеты и выгрузки
8. Аудит действий

## 2.2 Куратор

1. Главная панель куратора
2. Паспорт группы (10 вкладок)
3. Карточка студента
4. Посещаемость группы (сводно)
5. Успеваемость группы (сводно)
6. Уведомления группе и родителям
7. Документы группы
8. Риски и сигналы

## 2.3 Преподаватель

1. Главная панель преподавателя
2. Мои группы и дисциплины
3. Журнал оценок по предмету
4. Посещаемость по занятию
5. Комментарии и замечания
6. Материалы и задания
7. Ведомость по предмету

## 2.4 Студент

1. Главная панель студента
2. Мое расписание
3. Мои оценки
4. Моя посещаемость
5. Мои задания
6. Объявления
7. Уведомления

## 2.5 Родитель

1. Главная панель родителя
2. Карточка ребенка (ограниченная)
3. Успеваемость ребенка
4. Посещаемость ребенка
5. Сообщения от куратора
6. Подтверждение ознакомления

## 2.6 Администрация (директор/завуч/методист)

1. Дашборд по колледжу
2. Отчеты по группам
3. Отчеты по посещаемости
4. Отчеты по успеваемости
5. Реестр рисков

## 3. Паспорт группы (вкладки)

1. Общая информация
2. Состав группы
3. Успеваемость
4. Посещаемость
5. Дисциплина и замечания
6. Практика
7. Работа с родителями
8. Документы
9. События и объявления
10. Аналитика

## 4. Приоритет экранов для MVP-1

### P0 (обязательно)

1. Вход
2. Главная панель по роли
3. Паспорт группы (базовая версия)
4. Список студентов
5. Карточка студента (базовая)
6. Журнал оценок
7. Посещаемость
8. Уведомления
9. Кабинет студента (ограниченный)
10. Кабинет родителя (ограниченный)

### P1 (желательно)

1. Документы группы
2. Базовая аналитика рисков
3. Реестр задач куратора

### P2 (второй этап)

1. Расширенная аналитика
2. Массовые отчеты и выгрузки
3. Расширенная работа с практикой
