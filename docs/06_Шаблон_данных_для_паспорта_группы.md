# Шаблон данных для заполнения паспорта группы

Этот шаблон нужен для подготовки реальных паспортов ваших двух групп.

## 1. Общая информация по группе

1. Код/название группы
2. Специальность (код + название)
3. Курс
4. Период обучения (с/по)
5. Куратор (ФИО, контакт)
6. Староста (ФИО)
7. Статус группы (активна/выпуск/архив)

## 2. Состав группы (по каждому студенту)

1. ФИО
2. Дата рождения (опционально)
3. Контакт студента (телефон/email)
4. Родитель/законный представитель (ФИО, контакт, степень родства)
5. Статус обучения

## 3. Учебный блок

1. Перечень дисциплин
2. Преподаватель по каждой дисциплине
3. Текущая успеваемость (средний балл/долги)
4. Проблемные дисциплины

## 4. Посещаемость

1. Общее число пропусков
2. Уважительные / неуважительные
3. Студенты с превышением порога

## 5. Работа куратора

1. Проведенные беседы
2. Встречи с родителями
3. Индивидуальные меры поддержки

## 6. Документы группы

1. Перечень обязательных документов
2. Что уже загружено
3. Что отсутствует

## 7. События и объявления

1. Ближайшие мероприятия
2. Важные объявления
3. Подтверждение ознакомления

## 8. Риски

1. Список студентов в зоне риска
2. Причина риска
3. Назначенное действие
4. Ответственный
5. Срок контроля
